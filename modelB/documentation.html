This web application visualises wind power production forecasts for wind power plants across Europe.

Key Features

  • Interactive map with wind power plant information and predictions.
  • Customizable input fields to select turbine attributes and display predictions.
  • Time in Central European Time (CET/UTC+1), important when contributing data.
  • Built using Python, Shiny, and Jupyter notebooks for seamless functionality.

Model Details

  • Trained on a dataset of European wind power plants and ECMWF weather data.
  • Input features: Turbine Type, Hub Height, Capacity, Age, Wind Speed.
  • ML Model: Multi-Layer Perceptron (MLP) with four hidden layers and ReLU activation functions.
  • Missing Hub Height: Mean value from the database is used as default.
  • Missing Commissioning Date: Default set to June 2010 if missing.
  • Unknown Turbine Type: Auto-selects 'nan' if the type is not in the database.

Contribution Guidelines

Please contribute your data to help improve the model. Consider the following:

  • Ensure data is provided in kW for production values (see example file).
  • Consider the time zone (CET/UTC+1) when submitting data.
  • Extreme input values may lead to less accurate predictions due to limited training.

Motivation for Contribution

  • Helps improve the wind power forecast (WPF) model.
  • Facilitates wind power plant integration into the electricity grid.
  • Supports the reduction of dependency on fossil fuels.
  • Participation could be a Corporate Social Responsibility (CSR) initiative for sustainability.

Data Security and Privacy

  • All uploads are encrypted (HTTPS).
  • Data is stored securely until the next training cycle.
  • Data is destroyed after training.
  • Your data remain anonymous.

Transparency

Transparency is provided regarding how crowdsourced data is used to improve the model.
